The problem

At the Climathon event three problems were addressed:
  • How can you use existing open data sets to calibrate and check official reported emissions to better understand city emissions?
  • How can the data from ground-level emissions sensors be integrated and used to drive change?
  • Existing sensor systems are expensive, so can we build cheaper, open source sensor systems?

Our approach

Real time data, not annual reports

During the 24 hour hackathon we made a sensor that measured the CO2 level and reported this in real time to the cloud. We then retrieved this data through an open API and displayed it on a website.
